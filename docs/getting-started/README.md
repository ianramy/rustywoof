# Get Started with Rustywoof

Welcome to the official documentation for **Rustywoof**, the enterprise-grade secret scanner and supply chain watchdog. 

This guide will help you integrate Rustywoof into your local workflow and CI/CD pipelines to proactively detect exposed cryptographic credentials and compromised dependencies.

## Navigate the Introduction

If you are new to Rustywoof, we recommend following these steps:

1. :material-information-outline: **[Understand Rustywoof](what-is-rustywoof.md)**: Learn how our architecture compares to legacy tools and why we prioritize memory safety and speed.
2. :material-download: **[Install the Tool](installation.md)**: Choose your preferred package manager or operating system to get the CLI running locally.
3. :material-format-list-checks: **[Explore Features](features.md)**: Get a high-level overview of the available commands before diving into the deep technical references.
